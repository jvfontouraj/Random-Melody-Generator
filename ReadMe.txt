- Gerador de sequência melódica
	(escolher a escala ou cromática)
	(Selecionar a quantidade de notas)

.Input 1 -> Nota de referência (Fundamental) ou escala cromática
	Notas de referência = Escala Cromática, C, Cm, D, Dm, E, Em, F, Fm, G, Gm, A, Am, B, Bm

.Input 2 -> Escala Maior ou Menor (se cromática, desativar input)

.Input 3 -> Quantidade de notas (até 10)
	Notas = C, C#, D, D#, E, F, F#, G, G#, A, A#, B
